## About

lipas is currently in alpha. Source, examples, and roadmap live at
[lipas-otp/lipas](https://github.com/lipas-otp/lipas).

This blog is intentionally low-tech: just markdown files, rendered by
GitHub. No build step, no tracking, no comment system. For discussion,
open an issue on the main repo.
